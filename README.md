# ShopSense AI 🛒🤖

ShopSense AI is a shopping assistant web application built with Python and Streamlit. The application allows users to browse products, filter by category and budget, receive recommendations, and simulate an online shopping experience through an interactive interface.

## Features

* Product browsing and search
* Category and budget filtering
* Recommendation-based product suggestions
* Interactive shopping cart functionality
* User-friendly Streamlit interface

## Technologies Used

* Python
* Streamlit
* Pandas
* Git & GitHub

## Installation

```bash
git clone https://github.com/pvchaudhari24/shopsense-ai.git
cd shopsense-ai
pip install -r requirements.txt
streamlit run app.py
```
